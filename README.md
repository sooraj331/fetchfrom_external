A modern IPTV streaming application built with Flutter.
Supports M3U playlists, fullscreen playback, and smooth video streaming.

🚀 Features

📡 M3U playlist support

🔍 Channel search

🖼 Channel logo display

📂 Category-based channel grouping

📺 HLS (.m3u8) streaming support

🔄 Fullscreen landscape mode

🔋 Wakelock enabled during fullscreen playback

🎯 Lightweight and fast UI

📱 Android compatible

🛠 Built With

Flutter

Video Player

Chewie

Wakelock Plus

📌 Requirements

Android 7.0+

Valid Internet connection.
